# Android-usbview-python
View Android phone's display via ADB.

A python program which uses adb and screencap to sync your Android dispaly to desktop, uses multithread to increase fps.

## Dependencies

Python 3

matplotlab

Android Debug Bridge

Android 4.4+ (Client)
